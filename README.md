# AI Agent with MERN Stack

## Description
A web application built using the MERN (MongoDB, Express.js, React.js, Node.js) stack that implements AI agent functionality. Its a DEVIN-CLONE named Bevin. It generates code for you while you chat with your coworkers or friends. Make sure to add your own environment variables before running.

## Env Used

```
VITE_BACKEND_URL
PORT
MONGO_URI
JWT_SECRET
REDIS_HOST
REDIS_PORT=
REDIS_PASSWORD
FRONTEND_URL
GEMINI_API_KEY
```

## Features
- AI-powered interactions
- Full-stack implementation
- RESTful API endpoints
- Modern responsive interface

## Installation
1. Clone the repository
2. Install dependencies:
```bash
npm install
```

## Usage
- To start the frontend React+Vite server.
```bash
npm run dev
```
- To start the backend express server.
```bash
npx nodemon
```
